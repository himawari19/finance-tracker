================================================================================
                          DEPLOYMENT STEPS
================================================================================

1. Push ke GitHub:
   git remote add origin https://github.com/YOUR_USERNAME/keuangan-app.git
   git push -u origin main

2. Buat Project di Vercel:
   - Buka vercel.com
   - Import repository keuangan-app
   - Klik Import

3. Setup Vercel KV Database:
   - Di project, pilih Storage
   - Create Database â†’ KV
   - Pilih region
   - Klik Create

4. Deploy:
   - Klik Deploy
   - Tunggu hingga selesai
   - Buka URL yang diberikan

Lihat DEPLOYMENT.md untuk detail lengkap.

================================================================================
                          ENVIRONMENT VARIABLES
================================================================================

Untuk Vercel KV Database, tambahkan:

KV_URL=
KV_REST_API_URL=
KV_REST_API_TOKEN=
KV_REST_API_READ_ONLY_TOKEN=

(Vercel akan auto-generate ini saat membuat KV database)
